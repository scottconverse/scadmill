# Privacy

ScadMill has no telemetry.

The application makes no network requests except for the user-initiated or user-configured behaviors listed in specification A-5: AI-provider requests, the web engine artifact, an opt-in desktop update check, and explicit library or engine-version downloads. M0 implements none of those network paths.

The desktop OpenSCAD engine runs as a local subprocess. Source files and rendered geometry stay on the user's machine.
